# Next-Prompt

A simple NextJS AI Prompt app to manage AI prompts

# Tech Stack

Severless Arthitecture with only NextJS/React

- TypeScript
- NextJS & Next-auth (Google OAuth integrated)
- React
- TailwindCSS
- BCrypt
- MongoDB (saved to localhost)
- .env
